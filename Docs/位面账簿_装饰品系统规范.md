## 二、稀有度规则

### 2.1 四级稀有度定义

| 稀有度 | 标识色 | 定义标准 | 日常掉落权重 | 获取渠道 |
|--------|--------|---------|------------|---------|
| 普通 | 灰色 | 基础款，数量最多，快速积累用 | 70% | 日常掉落 / 代币购买 |
| 稀有 | 蓝色 | 有设计感，需要一定积累才能得到 | 25% | 日常掉落 / 代币购买 / 成就绑定 |
| 史诗 | 紫色 | 强风格感，含特效或动态元素 | 5% | 日常掉落（低概率）/ 成就绑定 |
| 传说 | 金色 | 唯一性强，有叙事意义 | 0%（不入随机池） | 仅成就绑定 / 连续里程碑 |

### 2.2 稀有度判定标准（新增装饰时参照）

**普通件**满足以下全部条件：
- 静态图，无粒子特效
- 风格通用，不强依赖特定世界观
- 视觉辨识度适中

**稀有件**满足以下至少一条：
- 有细节动画（如窗帘飘动、灯光闪烁）
- 强世界观风格（克苏鲁、古风、未来感明显）
- 与某个成就或书目有叙事关联

**史诗件**满足以下至少一条：
- 含粒子特效或全屏动态效果
- 打开/切换时有专属音效
- 改变主界面整体色调

**传说件**满足以下全部条件：
- 有专属解锁动画
- 与某个重要成就强绑定，有叙事文本
- 全服唯一来源（不重复投放）

## 三、获取规则

### 3.1 获取渠道优先级

```
传说件：成就绑定（唯一来源）
史诗件：成就绑定（主要）> 日常掉落5%（次要）
稀有件：日常掉落25% = 成就绑定 > 代币购买150枚
普通件：日常掉落70% > 代币购买50枚
```

### 3.2 代币购买规则

- 普通件：50枚，可直接在商店指定购买
- 稀有件：150枚，可直接在商店指定购买
- 史诗件：不可代币购买，必须通过成就或掉落获得
- 传说件：不可代币购买，必须通过成就获得

### 3.3 重复获取处理

- 已拥有的件再次掉落：转化为 10枚代币（普通）/ 20枚代币（稀有）/ 50枚代币（史诗）
- 传说件不会重复掉落

## 四、风格体系

每个槽位的装饰件按「风格」分类，风格是扩充内容时的主要维度。

### 4.1 已定义风格（初始版本）

| 风格ID | 风格名 | 视觉关键词 | 色调参考 |
|--------|--------|-----------|---------|
| STYLE-00 | 默认 | 简洁、中性、无强烈风格倾向 | 米白、原木 |
| STYLE-01 | 克苏鲁 | 触手、深海、眼球、孢子、低频震动 | 深紫、墨绿、污浊蓝 |
| STYLE-02 | 古风 | 云纹、锦缎、宣纸、宫灯、水墨 | 朱红、墨黑、金色 |
| STYLE-03 | 现代 | 简约、几何、工业感、都市 | 灰白、黑色、金属银 |
| STYLE-04 | 未来 | 全息、数据流、悬浮、光纤 | 电光蓝、赛博紫、冷白 |
| STYLE-05 | 奇幻 | 魔法阵、星图、水晶、精灵 | 星空蓝、翡翠绿、银白 |

### 4.2 风格扩充规则

新增风格时需定义：
1. 风格ID 和名称
2. 视觉关键词（3～5个词）
3. 色调参考
4. 该风格覆盖哪些槽位（不要求覆盖全部7个槽位）

**建议后续可扩充的风格方向：**
- 废土/蒸汽朋克
- 极简禅意（日式）
- 赛博都市霓虹（现代升级版）
- 黑暗哥特
- 童话森林

## 五、装饰品数据模板

每件装饰品需填写以下字段，新增时照此格式：

```
ID:          [槽位代码]-[风格代码]-[序号]，如 SLOT-01-F02-003
名称:         [位面风格命名]
槽位:         [SLOT-01 ～ SLOT-07]
风格:         [STYLE-00 ～ STYLE-XX]
稀有度:       [普通 / 稀有 / 史诗 / 传说]
含特效:       [是 / 否]，若是，注明特效类型
获取方式:     [日常掉落 / 代币购买XX枚 / 成就绑定(成就ID) / 里程碑绑定]
解锁描述:     [解锁时显示给玩家的一句话，位面风格]
视觉描述:     [给AI出图用的提示词方向，3～5句]
版本:         [首发 / vX.X 更新加入]
```

## 八、AI出图提示词方向（给自己备用）

每件装饰出图时建议包含以下要素：

**场景一致性关键词（所有件通用）：**
`室内场景局部特写, 精致插画风格, 无人物, 柔和光线, 高细节`

**各槽位专属关键词：**
- 账本：`书籍封面特写, 正面视角, 放置在桌面上`
- 窗帘：`室内窗帘, 从室内向窗户方向拍摄, 布料质感`
- 窗外景色：`透过窗户看到的风景, 窗框可见, 景深感`
- 地毯：`地毯俯视局部, 图案清晰, 质感丰富`
- 灯具：`室内吊灯仰视角度, 光晕效果, 背景虚化`
- 桌椅家具：`书桌正面视角, 桌面整洁, 椅子可见`
- 氛围特效：`透明背景粒子效果, PNG格式, 可叠加`
