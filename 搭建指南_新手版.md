# 位面账簿 Unity 搭建指南（新手版）

> 每一步都有截图级的操作描述，照着做就行。不确定的地方随时问我。

---

## 第一步：用 Unity Hub 创建项目

1. 打开 **Unity Hub**
2. **不要点 "New Project"**，而是点右上角 **"Add"（添加）** 按钮
3. 在弹出的文件夹选择器中，选择 `D:\PlaneLedger` 这个文件夹
4. Unity Hub 会识别到这个文件夹里有 Assets，自动把它当作项目添加
5. 如果它问你用哪个 Unity 版本打开，选你已安装的 **2021.3.x**
6. 双击项目打开它，**第一次打开会比较慢**（几分钟），等它编译完

> ⚠️ 如果 Unity Hub 不允许 "Add" 一个没有 ProjectSettings 的文件夹，那就：
> - 先用 Unity Hub 正常 **"New Project"** → 选 **2D** 模板 → 路径选 `D:\` → 项目名填 `PlaneLedger`
> - 它会创建完整的项目结构，你之前的 Assets 文件夹内容会保留

## 第二步：导入必要的包

项目打开后，先导入两个必须的包：

### 2.1 TextMeshPro
- Unity 通常会**自动弹窗**提示你导入 TMP Essentials
- 点 **"Import TMP Essentials"** 就行
- 如果没弹窗：菜单栏 → `Window` → `TextMeshPro` → `Import TMP Essential Resources`

### 2.2 Newtonsoft Json（JSON 序列化库）
- 菜单栏 → `Window` → `Package Manager`
- 左上角点 **"+"** 按钮 → 选 **"Add package by name..."**
- 输入：`com.unity.nuget.newtonsoft-json`
- 点 **"Add"**，等它安装完成

## 第三步：创建两个场景

### 3.1 创建 Boot 场景（启动场景）
1. 菜单栏 → `File` → `New Scene` → 选 **Basic** → 点 Create
2. 菜单栏 → `File` → `Save As`
3. 保存到 `Assets/_Project/Scenes/Boot.unity`

### 3.2 创建 MainPlane 场景（主场景）
1. 菜单栏 → `File` → `New Scene` → 选 **Basic** → 点 Create
2. 菜单栏 → `File` → `Save As`
3. 保存到 `Assets/_Project/Scenes/MainPlane.unity`

### 3.3 把两个场景加入 Build Settings
1. 菜单栏 → `File` → `Build Settings`
2. 点 **"Add Open Scenes"**（先打开 Boot 场景再点）
3. 再打开 MainPlane 场景，再次点 **"Add Open Scenes"**
4. **确保 Boot 在列表最上面（index 0）**，如果不是，拖动它到第一位
5. 关闭 Build Settings 窗口

## 第四步：搭建 Boot 场景

这个场景非常简单，只有一个 GameObject。

### 4.1 打开 Boot 场景
- 双击 `Assets/_Project/Scenes/Boot.unity` 打开

### 4.2 创建 GameManager 对象
1. 在 **Hierarchy（层级面板）** 中，右键 → `Create Empty`
2. 选中新建的对象，在右侧 **Inspector（检查器面板）** 最上方，把名字改为 `GameManager`
3. 点 Inspector 底部的 **"Add Component"** 按钮
4. 搜索 `GameManager`，选择 `PlaneLedger.Core.GameManager`
5. 在 Inspector 中你会看到：
   - `Auto Save Interval`：保持默认 `60`（每60秒自动保存）
   - `Main Scene Name`：填写 `MainPlane`（**必须和场景文件名完全一致**）

### 4.3 创建 SteamManager 对象（GameManager 的子对象）
1. 在 Hierarchy 中，**右键点击 GameManager** → `Create Empty`
2. 它会创建在 GameManager 下面（成为子对象），命名为 `SteamManager`
3. Add Component → 搜索 `SteamInitializer` → 添加

### 4.4 删除默认摄像机
- Boot 场景只负责启动，不需要 Camera
- 在 Hierarchy 中找到 `Main Camera`，右键 → Delete
- 同样删除 `Directional Light`（如果有的话）

### 4.5 保存
- `Ctrl + S` 保存场景

**此时 Boot 场景的 Hierarchy 应该长这样：**
```
Boot 场景
└── GameManager          ← 挂了 GameManager.cs
    └── SteamManager     ← 挂了 SteamInitializer.cs
```

## 第五步：搭建 MainPlane 场景（核心，最复杂）

### 5.1 打开 MainPlane 场景
- 双击 `Assets/_Project/Scenes/MainPlane.unity` 打开

### 5.2 设置摄像机
1. 在 Hierarchy 中找到 `Main Camera`
2. 在 Inspector 中确认：
   - `Projection` 是 **Orthographic**（2D 用正交摄像机）
   - `Size` 设为 `5`（之后根据美术调整）
   - `Background` 颜色可以改成你喜欢的深色，比如深蓝 `#1a1a2e`

### 5.3 创建背景
1. Hierarchy 右键 → `2D Object` → `Sprite` → `Square`
2. 命名为 `Background`
3. Inspector 中：
   - `Transform` → `Scale`：X=`20`, Y=`12`, Z=`1`（铺满屏幕的大矩形）
   - `Sprite Renderer` → `Color`：选个深色背景色
   - `Sprite Renderer` → `Sorting Layer`：（先保持 Default，后面再调）
   - `Order in Layer`：`-100`（确保在最后面）

### 5.4 创建装饰品根节点 + 7 个槽位

1. Hierarchy 右键 → `Create Empty` → 命名 `DecorationRoot`
2. 给 `DecorationRoot` 添加组件：Add Component → 搜索 `PlaneSpaceController` → 添加

3. 在 `DecorationRoot` 下面创建 7 个子对象（右键 DecorationRoot → Create Empty），分别命名为：

| 名称 | 含义 | 建议位置 (X, Y) |
|------|------|----------------|
| `Slot_Book` | 账本 | (0, -1) |
| `Slot_Curtain` | 窗帘 | (-2, 2) |
| `Slot_WindowView` | 窗外景色 | (-2, 2.5) |
| `Slot_Carpet` | 地毯 | (0, -3) |
| `Slot_Lamp` | 灯具 | (0, 3) |
| `Slot_Furniture` | 桌椅 | (0, 0) |
| `Slot_Ambient` | 氛围特效 | (0, 0) |

4. **给每个 Slot 子对象做以下操作**：
   - Add Component → `Sprite Renderer`（用来显示装饰品图片）
   - Add Component → 搜索 `DecorationSlotView` → 添加
   - 在 `DecorationSlotView` 组件中：
     - `Sprite Renderer` 字段：把自己身上的 Sprite Renderer 组件**拖进去**
       （方法：点 Sprite Renderer 字段右边的小圆圈，或者直接从 Inspector 中拖组件名）

5. **回到 DecorationRoot 上的 `PlaneSpaceController`**：
   - 你会看到 `Slots` 数组，`Size` 填 `7`
   - 把 7 个 Slot 子对象按顺序拖进去：
     - Element 0 = `Slot_Book`
     - Element 1 = `Slot_Curtain`
     - Element 2 = `Slot_WindowView`
     - Element 3 = `Slot_Carpet`
     - Element 4 = `Slot_Lamp`
     - Element 5 = `Slot_Furniture`
     - Element 6 = `Slot_Ambient`

### 5.5 创建交互入口点

1. Hierarchy 右键 → `Create Empty` → 命名 `InteractionPoints`

2. 在 `InteractionPoints` 下创建子对象：

**交互点1：账本（打开记账面板）**
- 右键 InteractionPoints → `2D Object` → `Sprite` → `Square`
- 命名 `IP_Book`
- Transform → Position: `(0, -1, 0)`，Scale: `(1.5, 1, 1)`
- Sprite Renderer → Color: 棕色 `#8B4513`（临时占位色）
- Add Component → `Box Collider 2D`（**必须有碰撞体才能点击**）
- Add Component → 搜索 `SceneInteractionPoint` → 添加
- 在 SceneInteractionPoint 中：
  - `Interaction Name` 填 `记账`
  - `On Clicked` 先留空（后面连接面板时再设置）

**交互点2：行李箱（打开装饰商店）**
- 同样的方式创建 `IP_Suitcase`
- Position: `(3, -2, 0)`
- Color: 灰色 `#666666`
- 同样加 Box Collider 2D + SceneInteractionPoint
- `Interaction Name` 填 `商店`

### 5.6 创建 Canvas_HUD（始终显示的信息）

1. Hierarchy 右键 → `UI` → `Canvas`
2. 命名为 `Canvas_HUD`
3. 在 Inspector 中设置 Canvas 组件：
   - `Render Mode`：**Screen Space - Overlay**
   - `Sort Order`：`0`
4. 找到自动添加的 `Canvas Scaler` 组件：
   - `UI Scale Mode`：选 **Scale With Screen Size**
   - `Reference Resolution`：X=`1920`, Y=`1080`
   - `Match`：拖到 `0.5`（宽高各半匹配）

5. Add Component → 搜索 `HUDController` → 添加

6. **在 Canvas_HUD 下创建文字**：

**生存天数文字：**
- 右键 Canvas_HUD → `UI` → `Text - TextMeshPro`
- 命名 `TXT_SurvivalDays`
- Rect Transform（在 Inspector 最上方）：
  - 点左上角的锚点方块 → 选 **Top-Left**（左上角对齐）
  - Pos X: `20`, Pos Y: `-20`
  - Width: `300`, Height: `40`
- TextMeshPro 组件：
  - Text: `生存天数: --`
  - Font Size: `24`
  - Color: 绿色

**代币文字：**
- 同样右键 Canvas_HUD → `UI` → `Text - TextMeshPro`
- 命名 `TXT_Token`
- 锚点：Top-Left
- Pos X: `20`, Pos Y: `-60`
- Width: `200`, Height: `40`
- Text: `代币: 0`
- Font Size: `24`
- Color: 琥珀色 `#E6B422`

7. **连接 HUDController 的引用**：
- 选中 `Canvas_HUD`
- 在 HUDController 组件中：
  - `Survival Days Text`：把 `TXT_SurvivalDays` 拖进去
  - `Token Text`：把 `TXT_Token` 拖进去

### 5.7 创建 Canvas_Panels（功能面板容器）

1. Hierarchy 右键 → `UI` → `Canvas`
2. 命名 `Canvas_Panels`
3. Canvas 设置同上（Screen Space - Overlay，Sort Order = `10`）
4. Canvas Scaler 设置同上（Scale With Screen Size, 1920x1080, Match 0.5）
5. Add Component → 搜索 `PanelManager` → 添加

> 这个 Canvas 下面将来会放记账面板、商店面板、成就面板等。
> 现在先创建一个空容器，面板内容后续逐个添加。

### 5.8 创建 Canvas_Popup（弹窗层）

1. Hierarchy 右键 → `UI` → `Canvas`
2. 命名 `Canvas_Popup`
3. Canvas：Sort Order = `100`（最高层，覆盖一切）
4. Canvas Scaler 设置同上

5. 在 Canvas_Popup 下创建 Toast 提示：
- 右键 Canvas_Popup → `UI` → `Panel`
- 命名 `Toast`
- Rect Transform：锚点 **Top-Center**
- Pos Y: `-80`, Width: `500`, Height: `60`
- Image 组件 → Color: 半透明黑 `(0, 0, 0, 180)`
- Add Component → `Canvas Group`
- Add Component → 搜索 `ToastNotification` → 添加

- 在 Toast 下创建 `UI` → `Text - TextMeshPro`
- 命名 `TXT_Toast`
- 铺满父对象（锚点选 Stretch-Stretch，上下左右 margin 都设 `10`）
- Alignment: Center + Middle
- Font Size: `20`
- Color: 白色

- 回到 Toast 的 ToastNotification 组件：
  - `Message Text`：拖入 `TXT_Toast`
  - `Canvas Group`：拖入自身的 Canvas Group 组件

### 5.9 创建右上角按钮组

在 `Canvas_HUD` 下继续添加：

1. 右键 Canvas_HUD → `Create Empty` → 命名 `ButtonGroup_TopRight`
2. Rect Transform → 锚点 **Top-Right**
3. Pos X: `-60`, Pos Y: `-80`

在 ButtonGroup_TopRight 下创建 3 个按钮：

**按钮1：统计**
- 右键 → `UI` → `Button - TextMeshPro`
- 命名 `BTN_Stats`
- Pos Y: `0`, Width: `100`, Height: `40`
- 子对象 Text: `统计`

**按钮2：成就**
- 命名 `BTN_Achievement`
- Pos Y: `-50`, 同尺寸
- Text: `成就`

**按钮3：设置**
- 命名 `BTN_Settings`
- Pos Y: `-100`, 同尺寸
- Text: `设置`

> 按钮的点击事件后面再连接。现在先搭好结构。

### 5.10 保存场景

`Ctrl + S`

**此时 MainPlane 场景的 Hierarchy 应该长这样：**

```
MainPlane 场景
├── Main Camera                          ← Orthographic
├── Background                           ← 大矩形Sprite，深色
├── DecorationRoot                       ← 挂了 PlaneSpaceController
│   ├── Slot_Book                        ← 挂了 DecorationSlotView + SpriteRenderer
│   ├── Slot_Curtain                     ← 同上
│   ├── Slot_WindowView                  ← 同上
│   ├── Slot_Carpet                      ← 同上
│   ├── Slot_Lamp                        ← 同上
│   ├── Slot_Furniture                   ← 同上
│   └── Slot_Ambient                     ← 同上
├── InteractionPoints
│   ├── IP_Book                          ← 挂了 SceneInteractionPoint + BoxCollider2D
│   └── IP_Suitcase                      ← 同上
├── Canvas_HUD (Sort Order: 0)           ← 挂了 HUDController
│   ├── TXT_SurvivalDays                 ← TextMeshPro, 左上角
│   ├── TXT_Token                        ← TextMeshPro, 左上角
│   └── ButtonGroup_TopRight
│       ├── BTN_Stats                    ← Button
│       ├── BTN_Achievement              ← Button
│       └── BTN_Settings                 ← Button
├── Canvas_Panels (Sort Order: 10)       ← 挂了 PanelManager
│   └── (后续添加各功能面板)
├── Canvas_Popup (Sort Order: 100)
│   └── Toast                            ← 挂了 ToastNotification + CanvasGroup
│       └── TXT_Toast                    ← TextMeshPro
└── EventSystem                          ← 创建Canvas时自动生成的，别删
```

## 第六步：运行测试

1. **确保当前打开的是 Boot 场景**（双击 Boot.unity）
2. 点 Unity 编辑器顶部的 **▶ Play** 按钮
3. 观察 **Console（控制台）** 窗口（`Window` → `General` → `Console`）
4. 你应该能看到类似这样的日志：
   ```
   [Steam] SteamInitializer 就绪（Steamworks.NET 待导入）
   [SaveSystem] 创建新存档
   [GameManager] 每日登录: 连续1天, 累计1天
   [GameManager] 初始化完成，加载主场景...
   ```
5. 场景会自动从 Boot 跳转到 MainPlane
6. 你应该能看到深色背景 + 左上角的 HUD 文字 + 右上角的按钮

## 常见问题

### Q: 脚本在 Add Component 里搜不到？
- 检查 Console 有没有**红色编译错误**，有错误时脚本不会显示
- 把错误截图给我

### Q: 运行后场景没跳转？
- 检查 Build Settings 里 Boot 场景是否在 index 0
- 检查 GameManager 的 `Main Scene Name` 是不是 `MainPlane`

### Q: Canvas 上的文字看不到？
- 确认 Canvas 的 Render Mode 是 Screen Space - Overlay
- 确认文字的 Color 不是透明的
- 确认 Font Size 够大（至少 20）

### Q: 点击场景物件没反应？
- 确认物件上有 **Collider2D**（Box Collider 2D）
- 确认场景里有 **EventSystem** 对象
- 确认摄像机上有 **Physics 2D Raycaster** 组件
  （如果没有：选中 Main Camera → Add Component → Physics 2D Raycaster）
