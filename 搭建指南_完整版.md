# 位面账簿 - 完整搭建指南（团结引擎/Unity）

> 本文档面向 Unity 零基础用户，每一步都告诉你点哪里、填什么。
> 适用于：团结引擎（Tuanjie Engine）或 Unity 2021.3.x LTS

---

## 第〇步：准备工作

### 0.1 安装团结引擎
1. 下载并安装 **Tuanjie Hub**（或 Unity Hub）
2. 在 Hub 中安装 **2021.3.x LTS** 版本
3. 安装时勾选：`Windows Build Support (IL2CPP)`

### 0.2 克隆项目代码
```bash
git clone git@github.com:361311733Raphaelia/PlaneLedger.git
```

### 0.3 在 Hub 中打开项目
1. 打开 Tuanjie Hub → 项目 → 添加
2. 选择克隆下来的 `PlaneLedger` 文件夹
3. 选择 2021.3.x 版本打开
4. **首次打开会很慢**（5-15分钟），耐心等待，看底部进度条

## 第一步：导入必要的包

打开后，菜单栏 → **Window（窗口）→ Package Manager（包管理器）**

### 1.1 TextMeshPro
1. Package Manager 左上角下拉选 **Unity Registry**
2. 搜索 `TextMeshPro`，点击 **Install**
3. 安装后弹窗问你要不要导入 TMP Essential Resources → 点 **Import**

### 1.2 Newtonsoft.Json
1. Package Manager 左上角点 **+** 号 → **Add package by name**
2. 输入：`com.unity.nuget.newtonsoft-json`
3. 点 **Add**

### 1.3 验证
Package Manager 左上角切换到 **In Project**，确认列表中有：
- TextMeshPro
- Newtonsoft Json

## 第二步：导入 CSV 数据为 ScriptableObject

### 2.1 等待编译完成
首次打开项目，Unity 会编译所有 C# 代码。看编辑器右下角的转圈图标，等它停下来。
如果 Console 窗口有红色错误，截图给我看。

### 2.2 运行 CSV 导入工具
1. 菜单栏 → **PlaneLedger → CSV 导入工具**
2. 点击 **全部导入**
3. 底部会显示："装饰品导入完成：新建 XX"、"卡片导入完成：新建 XX"、"成就导入完成：新建 XX"

### 2.3 验证
在 Project 窗口中展开：
- `Assets/_Project/ScriptableObjects/Decorations/` → 应该有 ~89 个 .asset 文件
- `Assets/_Project/ScriptableObjects/WisdomCards/` → 应该有 89 个 .asset 文件
- `Assets/_Project/ScriptableObjects/Achievements/` → 应该有 164 个 .asset 文件

## 第三步：创建 Boot 场景

### 3.1 创建场景文件
1. Project 窗口 → 右键 `Assets/_Project` → **Create → Folder** → 命名 `Scenes`
2. 右键 `Scenes` 文件夹 → **Create → Scene** → 命名 `Boot`
3. 双击 `Boot` 场景打开它

### 3.2 创建 GameManager 对象
1. Hierarchy 窗口 → 右键 → **Create Empty** → 命名 `GameManager`
2. 选中 `GameManager` 对象 → Inspector 窗口 → **Add Component**
3. 搜索 `GameManager` → 选中 `PlaneLedger.Core.GameManager`
4. 在 Inspector 中设置：
   - `Auto Save Interval`: 60
   - `Main Scene Name`: MainPlane

### 3.3 删除多余对象
- 删除场景中默认的 `Main Camera` 和 `Directional Light`（Boot 场景不需要）

### 3.4 保存场景
Ctrl+S 保存

## 第四步：创建 MainPlane 场景

### 4.1 创建场景
1. Project 窗口 → 右键 `Assets/_Project/Scenes` → **Create → Scene** → 命名 `MainPlane`
2. 双击打开

### 4.2 创建主摄像机
默认已有 `Main Camera`，保留它。设置：
- Position: (0, 0, -10)
- Projection: Orthographic
- Size: 5
- Background: 深蓝色（#1A1A2E 或类似暗色）

### 4.3 创建 DatabaseManager
1. Hierarchy → 右键 → **Create Empty** → 命名 `DatabaseManager`
2. Add Component → 搜索 `DatabaseManager` → 选 `PlaneLedger.Data.DatabaseManager`
3. 在 Inspector 中，你会看到三个数组槽：
   - `All Decorations`
   - `All Cards`
   - `All Achievements`
4. **批量拖入 SO 文件**（关键步骤）：
   - 在 Project 窗口进入 `Assets/_Project/ScriptableObjects/Decorations/`
   - 按 `Ctrl+A` 全选所有 .asset 文件
   - 拖到 Inspector 中 `All Decorations` 那个数组槽上（拖到数组名字上，不是里面）
   - 同样操作 WisdomCards → `All Cards`
   - 同样操作 Achievements → `All Achievements`

### 4.4 创建 UI Canvas 结构

**重要概念**：Unity 的 UI 需要放在 Canvas 里面。我们创建 3 个 Canvas 分层管理。

#### 4.4.1 创建 Canvas_HUD（常驻顶部信息）
1. Hierarchy → 右键 → **UI → Canvas** → 命名 `Canvas_HUD`
2. 选中它 → Inspector → Canvas 组件 → `Sort Order`: 0
3. Canvas Scaler 组件：
   - UI Scale Mode: **Scale With Screen Size**
   - Reference Resolution: 1920 x 1080
   - Match: 0.5

##### 在 Canvas_HUD 下创建子对象：

**生存天数文字**：
1. 右键 Canvas_HUD → **UI → Text - TextMeshPro** → 命名 `Text_SurvivalDays`
2. RectTransform：
   - Anchor Preset: 左上角（按住Alt点击左上角那个预设）
   - Pos X: 150, Pos Y: -30
   - Width: 250, Height: 40
3. TextMeshPro 组件：
   - Text: `生存天数: --`
   - Font Size: 24
   - Color: 绿色 (0.2, 0.8, 0.3)

**代币文字**：
1. 右键 Canvas_HUD → **UI → Text - TextMeshPro** → 命名 `Text_Token`
2. 同样锚定左上角，Pos X: 150, Pos Y: -70, Width: 200, Height: 40
3. Text: `代币: 0`, Font Size: 24, Color: 琥珀色 (0.9, 0.7, 0.2)

**HUD 控制器**：
1. 选中 `Canvas_HUD` → Add Component → 搜索 `HUDController`
2. 把 `Text_SurvivalDays` 拖到 `Survival Days Text` 槽
3. 把 `Text_Token` 拖到 `Token Text` 槽

#### 4.4.2 创建 HUD 功能按钮

在 Canvas_HUD 下创建 8 个按钮（右键 Canvas_HUD → UI → Button - TextMeshPro）：

| 按钮名 | 文字 | 位置建议（右侧竖排）|
|--------|------|---------------------|
| Btn_DailyMatrix | 日常记账 | Anchor右侧, PosX:-100, PosY:-50 |
| Btn_MonthlyOverview | 月度统计 | PosY:-100 |
| Btn_Amortization | 年度平摊 | PosY:-150 |
| Btn_Settlement | 每日结算 | PosY:-200 |
| Btn_DecorationShop | 装饰商店 | PosY:-280 |
| Btn_CardCollection | 残章收藏 | PosY:-330 |
| Btn_Achievement | 成就列表 | PosY:-380 |
| Btn_Settings | 设置 | PosY:-430 |

每个按钮：Width: 160, Height: 40, Font Size: 18

#### 4.4.3 创建 Canvas_Panels（功能面板层）
1. Hierarchy → 右键 → **UI → Canvas** → 命名 `Canvas_Panels`
2. Sort Order: 10（在HUD之上）
3. Canvas Scaler：同上（Scale With Screen Size, 1920x1080, Match 0.5）

##### 创建面板结构（每个面板是一个子对象）：

对于下面的每一个面板，都按这个模式创建：

**通用面板创建步骤**（以 DailyMatrix 为例）：
1. 右键 Canvas_Panels → **UI → Panel** → 命名 `Panel_DailyMatrix`
2. RectTransform: Stretch All（Anchor 四角撑满父对象）或设为中心定位 Width:800 Height:600
3. Image 组件：Color 设为半透明深色 (0.1, 0.1, 0.15, 0.95)
4. Add Component → 搜索对应脚本名

需要创建的面板列表：

| 面板名 | 脚本 | Panel Id（在Inspector中填）|
|--------|------|---------------------------|
| Panel_DailyMatrix | DailyMatrixPanel | DailyMatrix |
| Panel_MonthlyOverview | MonthlyOverviewPanel | MonthlyOverview |
| Panel_Amortization | AmortizationPanel | Amortization |
| Panel_Settlement | SettlementPanel | Settlement |
| Panel_DecorationShop | DecorationShopPanel | DecorationShop |
| Panel_CardCollection | CardCollectionPanel | CardCollection |
| Panel_AchievementList | AchievementListPanel | AchievementList |
| Panel_Settings | SettingsPanel | Settings |
| Panel_DropResult | DropResultPanel | DropResult |
| Panel_ConfirmDialog | ConfirmDialog | ConfirmDialog |
| Panel_InputAmount | InputAmountDialog | InputAmount |
| Panel_CategorySelector | CategorySelector | CategorySelector |

**每个面板的内部子对象**：参见下方详细表格。

#### 4.4.4 创建 Canvas_Toast（消息提示层）
1. 创建新 Canvas → 命名 `Canvas_Toast`
2. Sort Order: 100（最上层）
3. Canvas Scaler 同上
4. 右键 Canvas_Toast → UI → Text - TextMeshPro → 命名 `Text_Toast`
5. 设置：居中，Width: 600, Height: 50, Font Size: 22, Color: 白色
6. 选中 Canvas_Toast → Add Component → ToastNotification
7. 把 `Text_Toast` 拖到脚本的槽位中（查看 ToastNotification 的 SerializeField）

## 常见问题

### Q: Console 有红色报错怎么办？
截图给我看，我帮你修。常见原因：
- 包没装：检查 TextMeshPro 和 Newtonsoft.Json
- SO 没导入：重新运行 CSV 导入工具
- 引用丢失：Inspector 中有黄色 Missing 提示，重新拖拽

### Q: 面板打开是空白的？
面板内部的子对象（Text、Button等）没创建或没拖到脚本槽位。检查 Inspector 中是否有 `None` 的引用。

### Q: 预制体怎么创建？
从 Hierarchy 中把对象拖到 Project 窗口的文件夹里，就自动变成预制体了。

### Q: 拖拽引用是什么意思？
就是在 Inspector 面板中，把 Hierarchy 中的对象拖到脚本组件的对应槽位上。槽位显示为一个带圆点的框，旁边有字段名。

### Q: 团结引擎的菜单是中文吗？
是的，如果你用团结引擎，上面提到的英文菜单会自动变成中文，对照：
- Window → 窗口
- Package Manager → 包管理器
- Create → 创建
- Scene → 场景
- UI → UI
- Canvas → 画布
- Panel → 面板
- Button → 按钮
- Text → 文本
- Add Component → 添加组件
- Build Settings → 构建设置
